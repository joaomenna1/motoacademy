Atributos das listas

  1. reversed - indica que ira ter uma lista reversa
  2. start - indica por qual elemento deve começar a numeração start="10"
  3. type
    • 'a' - indica letras minusculas
    • 'A' - indica letras maisculas
    • 'i' - indica algarismo romanos minusculos
    • 'I' - indica algarismo romanos maiusculos
    • '1' - indica números (padrão)
